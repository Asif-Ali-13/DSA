----------------------------------------------------------------------------------------

problem statement:
Given the head of a linked list, 
remove the nth node from the end of the list and return its head.

----------------------------------------------------------------------------------------

approach:
    1. count the size of the ll.
    2. add a dummy node to the head of the ll.
    3. traverse the ll from the dummy node to size-n.
    4. remove the node at size-n.

----------------------------------------------------------------------------------------

code:
ListNode removeNthFromEnd(ListNode head, int n) {
    int size = 1;
    ListNode count = head;
    while(count.next != null){
        count = count.next;
        size++;
    }
    ListNode st = new ListNode(0, head);
    ListNode temp = st;
    for(int i=0; i<(size-n); i++){
        temp = temp.next;
    }
    if(temp.next != null) temp.next = temp.next.next;
    return st.next;
}

----------------------------------------------------------------------------------------

tc: O(N)
sc: O(1)
