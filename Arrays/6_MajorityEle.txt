----------------------------------------------------------------------------------------

problem statement: 
Given an array nums of size n, return the majority element.
The majority element is the element that appears more than ⌊n / 2⌋ times. 
You may assume that the majority element always exists in the array.

----------------------------------------------------------------------------------------

approach: 
    1. freq of majority ele > n/2
    2. if cnt = 0 -> consider ele and cnt = 1
    3. if ele -> cnt++ else cnt--

----------------------------------------------------------------------------------------

code: 

    int cnt = 0, ele = -1;
    for(int i: nums) {
        if(cnt == 0){
            ele = i;
            cnt = 1;
        }
        else if (i == ele) cnt++;
        else cnt--;
    }
    return ele;

----------------------------------------------------------------------------------------

tc: O(N)
sc: O(1)
