----------------------------------------------------------------------------------------

problem statement: 
Given an integer array nums, move all 0's to the end of it 
while maintaining the relative order of the non-zero elements.

----------------------------------------------------------------------------------------

approach:
    1. init j = -1  (for tracking the index of zero)
    2. after traversing if j != -1
    3. swap the zero with non - zero elements to move it to end

key points: 
    1. j = -1
    2. loop i (j+1 to n)
    3. if(a[i] != 0) swap i and j th elements

----------------------------------------------------------------------------------------

code : 

int j = -1, n = nums.length; 

for(int i=0; i<n; i++){
    if(nums[i] == 0){
        j = i;
        break;
    }
}

if(j == -1) return;

for(int i=j+1; i<n; i++){
    if(nums[i] != 0){
        swap -> ith and jth elements
    }
}

----------------------------------------------------------------------------------------

tc: O(N)
sc: O(1)
