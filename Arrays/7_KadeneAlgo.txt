----------------------------------------------------------------------------------------

problem statement: 
Given an integer array nums, find the subarray with the largest sum, and return its sum.

follow up : can you also print the subarray

----------------------------------------------------------------------------------------

code: 
    int n = nums.length;
    int sum = 0, res = nums[0];
    int st = -1, ed = -1;
    for(int i=0; i<n; i++){
        sum += i;
        if(sum >= res){
            res = sum;
            ed = i;
        }

        if(sum < 0){
            sum = 0;
            st = i;
        }
    }
    for(int i=st+1; i<=ed; i++){
        System.out.print(nums[i] + " ");
    }
    return res;

----------------------------------------------------------------------------------------

tc: O(N)
sc: O(1)
