----------------------------------------------------------------------------------------

problem statement: 
Given a array which is rotated k times and you need to find out 
whether it was a sorted array or not before rotation.

----------------------------------------------------------------------------------------

approach:   
    1. find out the number of inversions 
    2. inversions (nums[i] > nums[i+1])
    3. also consider nums[0] < nums[len-1]
    4. if inversions <= 1 -> true (sorted)

----------------------------------------------------------------------------------------

tc: O(N)
sc: O(1)
