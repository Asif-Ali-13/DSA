Time complexity : 

for sorted map : (Tree map in java)
    all cases -> best, avg., worst -> O(logN)

for unordered map : (HashMap in java)
    best, avg. -> O(1)
    worst -> O(N) -> bcz of internal collisions 

where N -> no. of element in the map
