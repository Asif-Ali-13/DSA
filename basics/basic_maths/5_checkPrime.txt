we are given a number N
we want to check if N is prime or not

----------------------------------------------------------------------------------------

Basic approach:

if(n == 1) return false;
if(n == 2) return true;
if(n%2 == 0) return false;

for(int i=3; i<=Math.sqrt(n); i+=2){
    if(n%i == 0) return false;
}
return true;

tc: O(sqrt(n))
sc: O(1)
