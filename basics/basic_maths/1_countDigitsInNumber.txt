let say N = 29493479

we want to find the number of digits in N

----------------------------------------------------------------------------------------

Basic approach:

int count = 0;
while(n > 0){
    n = n/10;
    count++;
}
return count;

tc: O(log10(n) + 1)
sc: O(1)

----------------------------------------------------------------------------------------

optimal approach:

return int(Math.log10(n)+1);

or

return String.valueOf(n).length();

tc: O(1)
sc: O(1)

----------------------------------------------------------------------------------------
