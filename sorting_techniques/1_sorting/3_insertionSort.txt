----------------------------------------------------------------------------------------

approach: 
    1. start i from idx 1 to n-1
    2. from i to 0
    3. compare and swap

key points: 
    1. loop i (1 to n-1)
    2. j = i
    3. while j > 0 and arr[j-1] > arr[j] -> swap

----------------------------------------------------------------------------------------

code: 

int n = arr.length;
for(int i=1; i<n; i++){
    int j =i;

    while(j>0 && arr[j-1] > arr[j]){
        int temp = arr[j];
        arr[j] = arr[j-1];
        arr[j-1] = temp;
        j--;
    }
}

----------------------------------------------------------------------------------------

tc: O(N^2)
sc: O(1)
